# Team 1710 2026 Robot Code
[![CI](https://github.com/FRC-Team-1710/2026-Robot/actions/workflows/main.yml/badge.svg)](https://github.com/FRC-Team-1710/2026-Robot/actions/workflows/main.yml)
This repository holds all code for Team 1710's 2026 Robot code for the game Rebuilt. 

We are a Java team that uses PhotonVision, Epilogue, Phoenix Pro, and a custom Autonomous Builder. On our robot we are using a RoboRIO 2, Swerve Mk5, Orange Pi's, and a Luma P1. Special thanks to 5712 for their [template](https://github.com/Hemlock5712/2026-Template#).
